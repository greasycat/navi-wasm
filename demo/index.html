Navi WASM demo

Six chart types in Rust + WASM

Scatter, line, bar, heatmap, rooted tree, and network/DAG all rendered on HTML5 Canvas via Rust and WebAssembly. The two graph explorers now use astronomy-themed datasets to show flexible node and edge styling from the frontend, and the ordered-tree variant keeps sibling sequence explicit across two descendant bands.
